什么是 Rocky？

Rocky 是一个开源 AI 编程代理，帮助你在终端、IDE 或桌面端编写代码。它读代码、改代码、跑测试，实时流式展示推理过程。事情顺利时他说 amaze!，困惑时说 i no know! i learn!

  • 思维链推理 — 实时展示推理过程，透明可解释
  • 多模型支持 — DeepSeek、Claude、GPT、Gemini 等 75+ 提供商
  • LSP 集成 — 自动加载合适的语言服务器，实时诊断
  • 多会话并行 — 同一项目启动多个代理同时工作
  • 记忆系统 — 跨会话持久记忆，文件即真理
  • 梦境系统 — 离线整理对话，零 API 费用
  • 基准测试 — SWE-bench Verified 打分，数字说话
  • 零迁移成本 — 直接读取 Claude Code / Codex 已有配置
  • MCP 支持 — 自动发现 MCP 服务，扩展工具箱
  • 技能系统 — 按需加载自定义技能，按需使用

隐私优先的设计

Rocky 不存储任何代码或上下文数据。所有记忆以纯 Markdown 文件存储在你本地 .rockycode/memory/，你拥有完全控制权。 了解更多关于隐私。

常见问题

Rocky 是一个交互式 AI 编程智能体，名字来自《挽救计划》里的外星工程师。他能在你的项目里读代码、改代码、跑命令，并实时流式展示推理过程。
安装 Python 3.11+ 和 uv，克隆仓库，uv sync，配置 .env 中的 API Key，然后 uv run rockycode chat 即可启动。
Rocky 本身免费开源，但你需要一个 OpenAI 兼容的 API Key（默认指向 DeepSeek）。你也可以使用自己的 Claude、GPT 或 Gemini API Key。
Rocky 目前以终端界面 (TUI) 为主。社区正在开发桌面应用和 IDE 扩展。
Rocky 完全免费，MIT 开源协议。你只需支付所使用模型提供商的 API 费用。
Rocky 将所有数据保存在你的本地机器上。记忆文件是纯 Markdown，你随时可以查看、编辑或删除。代码和对话数据不会上传到任何 Rocky 服务器。
是的，Rocky 采用 MIT 协议，100% 开源。你可以在 GitHub 上查看、修改和贡献代码。
